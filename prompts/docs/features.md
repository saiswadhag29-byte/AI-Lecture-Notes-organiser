# AI Lecture Notes Organizer

This application converts unstructured lecture notes into structured and easy-to-study notes using AI.

## Features

### 1. Automatic Note Organization
- Converts messy notes into clear headings and bullet points
- Groups related topics together

### 2. Importance Marking
- ⭐⭐⭐ Very important: definitions, key concepts, formulas
- ⭐⭐ Medium importance: examples and explanations
- ⭐ Low importance: additional details

### 3. Exam-Focused Mode
- Shows only important and exam-relevant content
- Helps in quick revision before exams

### 4. Beginner-Friendly Mode
- Uses simple language
- Adds small real-life examples for better understanding

### 5. Summary Generator
- Generates a short summary at the end of notes
- Useful for last-minute revision
